# Sovereign Contracts Group Website

Website for [sovereign-contracts.com](https://sovereign-contracts.com), hosted on GitHub Pages.

## Deployment

1. Create a new GitHub repository (e.g., `sovereign-contracts-website`)
2. Push the contents of this folder to the `main` branch
3. Go to **Settings > Pages** and set source to `main` branch, root folder
4. Add custom domain `sovereign-contracts.com` in Pages settings
5. Configure DNS: add a CNAME record pointing `sovereign-contracts.com` to `<username>.github.io`

## Files

- `index.html` — Full single-page website
- `favicon.ico` — Browser tab icon
- `apple-touch-icon.png` — iOS home screen icon
- `CNAME` — Custom domain config for GitHub Pages
